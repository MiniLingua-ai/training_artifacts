# Evaluation

This folder contains evaluation artifacts for the **MiniLingua 1B** model.  

## Benchmarks
- **MMMLU-X** – Multilingual Massive Multitask Language Understanding  
- **Belebele** – Multilingual Open-book Question Answering Task.
- **FLORES-200** – Multilingual translation benchmark  
- **SIB** – Multilingual Topic Classification Dataset.
- **MassiveSum** – Multilingual Summarisation Dataset.
